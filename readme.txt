== Installation ==

1. Upload the `chat-bubble` directory to the `/wp-content/plugins/` directory via FTP
2. Alternatively, upload the `chat-bubble.zip` file to the 'Plugins->Add New' page in your WordPress admin
area
3. Activate the plugin through the 'Plugins' menu in WordPress
